ENG:
Write a program that allows you to adjust the brightness of LEDs 1/2/3/4 in a blue LED module using an encoder. Use events to control the encoder. Only one LED is lit at a time. Selecting a LED from the module is done using buttons. The encoder only changes the intensity. Display the current intensity value on the console.


PL:
Napisz program, który pozwala regulować przy pomocy enkodera jasność świecenia diody LED 1/2/3/4 w module niebieskich diod świecących. Do obsługi enkodera wykorzystaj zdarzenia (events). W danym czasie świeci się tylko jedna dioda LED. Wybór diody z modułu odbywa się poprzez przyciski. Enkoder zmienia jedynie intensywność świecenia. Proszę na konsoli wyświetlać aktualną wartość intensywności.
